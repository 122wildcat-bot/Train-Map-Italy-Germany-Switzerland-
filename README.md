# Train Map — Italy, Switzerland & Germany

An interactive, single-file rail map of Italy, Switzerland, and Germany.

## Viewing

Open `index.html` in any modern web browser. The page is fully self-contained
(HTML, CSS, and JavaScript in one file) — no build step or server required.

### GitHub Pages

To publish it online, enable GitHub Pages for this repository (Settings →
Pages → Source: `main` branch, root folder). The map will then be available at
the published Pages URL.
